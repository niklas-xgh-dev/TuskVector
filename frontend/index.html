Welcome to the TuskVector! Our API allows you to manage vector data with ease and query an LLM with it in the most effective way (RAG) - what's a RAG?.

Whether you're working on ML projects, data analysis, or any application that requires efficient vector operations - this lightweight API framework could be a fit for you.

Our current version is capable of:

  • Store embedded data
  • Perform vector similarity searches
  • Use vector search results to enrich LLMs (that's the RAG part)
  • Integrate RAG capabilities into your applications
